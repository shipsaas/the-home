# The Home - ShipSaaS

The Home of ShipSaaS.

## Tech

- Vue 3 x TypeScript
- TailwindCSS

## Development 🧰

Note: this project is using [`pnpm`](https://pnpm.io/)

```bash
pnpm install
pnpm start
```

## Production 🚀

```bash
# build and preview
pnpm preview

# build only
pnpm build
```
